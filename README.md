# Offria — Private Local AI Assistant

Offria（オフリア）はローカルAIを利用したプライベートPWAチャットアプリです。

## Demo

実際に利用できます。
https://floral-mouse-902c.224096.workers.dev/

## Features

- ローカルAIチャット
- OCR文字認識
- QRコード読み取り
- 計算モード
- PWA対応
- オフライン動作対応

## Technology

- HTML / CSS / JavaScript
- Cloudflare Pages
- WebAssembly
- GGUFモデル
