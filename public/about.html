Offria — Private Local AI Assistant 技術紹介

概要

Offria — Private Local AI Assistantはスマートフォンのブラウザ上で動作する ローカルAIチャットPWAです。 WebAssemblyとGGUF形式のAIモデルを利用し、 端末内でAI処理を実行します。

主な技術

  • HTML / CSS / JavaScript
  • PWA（Progressive Web App）
  • WebAssembly
  • GGUF量子化AIモデル
  • Cloudflare Pages

搭載機能

  • ローカルAIチャット
  • OCR文字認識
  • QRコード・バーコード読み取り
  • 計算モード
  • オフライン利用
